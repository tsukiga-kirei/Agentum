# 前端设计规范

## 1. 设计目标

Agentum 前端不是营销页面，而是一个专业的企业工作台。

目标：

- 清晰
- 稳定
- 美观
- 高信息密度但不压迫
- 易于长期使用

## 2. 技术选择

推荐：

- React
- TypeScript
- React Flow
- Tailwind CSS
- Radix UI / shadcn/ui
- lucide-react
- TanStack Query
- Zustand
- React Hook Form
- Zod

## 3. 页面结构

推荐核心页面：

- 业务工作台首页
- 我的待办
- 可用流程模板
- 发起流程
- 工作流列表
- 工作流编辑器
- 工作流运行详情
- 智能体模板库
- Skills 库
- MCP 服务库
- 知识库
- 交付目标管理
- 审计日志
- 权限管理
- 系统设置

前端页面需要区分产品区域：

- **业务使用区**：面向业务人员，不默认展示画布。
- **流程设计工作台**：面向流程设计者，提供完整画布和节点配置。
- **能力管理区**：面向能力管理员，管理智能体、Skills、MCP、知识库和交付目标。
- **系统管理区**：面向管理员，管理用户、角色、空间、凭证和系统配置。
- **运行审计区**：面向管理员和审计人员，查看运行证据链。

详细分区见 [product-surfaces.md](./product-surfaces.md)。

## 4. 画布要求

画布是产品核心，优先保证可读性和稳定性。

要求：

- 节点类型颜色统一。
- 节点尺寸稳定，不因文字长短大幅跳动。
- 连线清晰，选中态明显。
- 支持缩放、拖拽、框选、自动布局。
- 支持节点搜索和快速定位。
- 支持运行态高亮当前节点。
- 支持查看节点输入输出变量。
- 支持查看每个步骤调用的 Skills、MCP、知识库和交付目标。
- 支持明确区分一次性输出、追问确认、分析后暂停三种智能体输出模式。
- 支持并行分组的进度展示，例如 `2 / 3 已完成`。
- 支持从暂停提示直接定位到需要用户输入或审核的节点。

节点颜色建议：

| 节点类型 | 颜色 |
| --- | --- |
| 触发节点 | 中性灰 |
| 用户输入节点 | 橙色 |
| 智能体节点 | 蓝色 |
| 并行节点组 | 绿色 |
| 合并节点 | 紫色 |
| 条件分支节点 | 黄色 |
| 人工审核节点 | 红色 |
| 交付节点 | 橘红色 |

## 5. 节点卡片

节点卡片必须展示：

- 节点名称
- 节点类型
- 配置摘要
- 输入变量
- 输出变量
- 暂停标记
- 运行状态
- 输出模式
- 工具调用数量
- 是否允许追问
- 是否需要用户输入
- 是否属于并行分组

不要在节点卡片里堆太多配置项，详细配置放右侧面板。

## 6. 右侧配置面板

节点配置使用右侧抽屉。

推荐结构：

```text
基础信息
输入变量
核心配置
能力装配
交互模式
输出变量
权限与审计
高级设置
```

要求：

- 高级设置默认折叠。
- 变量选择器支持点击插入。
- 错误提示要具体。
- 表单保存前必须校验。
- 修改生产流程时要提示影响范围。
- 智能体节点必须展示已装配 Skills、MCP、知识库、模型和输出模式。
- 用户输入节点必须展示字段清单、输出变量、等待对象和超时策略。
- 并行节点组必须展示完成策略、超时策略和子任务列表。

## 7. 运行态界面

运行态必须清楚告诉用户当前流程在哪里、为什么停住、下一步谁处理。

应展示：

- 当前状态
- 当前节点
- 节点耗时
- 等待对象
- 暂停原因
- 输入变量
- 输出变量
- 错误信息
- 重试按钮
- 交付物
- 每个节点完成的工作摘要
- 工具调用记录
- Skills / MCP 使用记录
- 智能体追问记录
- 用户输入记录
- 并行任务完成进度
- 变量快照

运行态页面应提供底部详情面板，使用 `执行日志`、`变量快照`、`工具调用`、`用户输入`、`审核记录`、`交付记录` 等标签页承载证据链。

更详细的工作流展示规范见 [frontend-workflow-visualization.md](./frontend-workflow-visualization.md)。

## 8. 视觉要求

要求：

- 不使用大面积无意义渐变。
- 不用花哨动效掩盖信息层级。
- 卡片圆角保持克制。
- 按钮尺寸和间距统一。
- 图标优先使用 lucide-react。
- 表格、表单、抽屉、弹窗风格统一。
- 移动端可以只保证基础查看，不强求完整画布编辑。

## 9. 文案要求

文案使用中文，简洁具体。

推荐：

- “智能体节点缺少输出变量”
- “当前流程正在等待审核人处理”
- “该 MCP 服务需要管理员授权后才能调用”

不推荐：

- “配置错误”
- “系统异常”
- “操作失败”
