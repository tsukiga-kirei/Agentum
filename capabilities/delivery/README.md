# 交付能力实现

本目录存放交付能力的适配器、模板和本地验证材料。

当前阶段仅保留邮箱发送交付能力；Word / PDF 生成、OA 创建流程、IM 推送、Webhook 回调、数据库写入等后续再扩展。

建议目录：

```text
<delivery-key>/
  templates/   文档、邮件或消息模板
  src/         适配器源码或脚本
  tests/       模板渲染、参数校验、失败重试和脱敏测试
  README.md    本地验证和部署说明
```

高风险交付能力必须经过权限校验、审批或二次确认，并写入交付记录和审计日志。

已提供能力材料：

- `email-smtp/`：SMTP 邮箱交付配置清单，可配合本地 Mailpit 验证。
